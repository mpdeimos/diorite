Diorite Library
===============

_Private utility and widget library for Nuvola Player project based on GLib, GIO and GTK._

Status: Early alpha
--------------------

  - No API & ABI stability
  - Dependencies are not frozen
  - No test case
  - No documentation
  - Not suitable for your project

Dependencies
------------

  - Python 2 to run ./waf
  - Python 3 to run ./testgen.py
  - glib-2.0 2.34
  - ghread-2.0 2.34
  - gio-2.0 2.38
  - gtk+-3.0 3.4

Build
-----

    $ ./waf configure
    $ ./waf build

Install
-------

    # ./waf install
    
Uninstall
---------

    # ./waf uninstall
